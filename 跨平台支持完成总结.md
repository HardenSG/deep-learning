## 🚀 Mac上使用（超简单）

### 方法1: 一键启动（推荐）

```bash
# 第1步：安装
chmod +x install_mac.sh
./install_mac.sh

# 第2步：启动
./start_mac.sh
```

### 方法2: 手动启动

```bash
# 安装
pip3 install streamlit plotly

# 启动
streamlit run app.py
```
